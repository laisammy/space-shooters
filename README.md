# Space Shooters (WORK IN PROGRESS)
## Made for Hack Club's Resolution Game Dev Pathway
This is a third person shooter game made in Godot 4.6.1. You play as a fighter jet in space shooting enemy jets and asteroids down.

Week 8's features:
- Tie Fighter behaviour & hitbox
- Asteroids
- Spawning of tie fighters and asteroids

Week 7's features:
- Guns and lasers
- Shooting function
- Tie fighter (Enemy) (WIP)

Week 6's features:
- Player movement and camera
- Space environment
- Simple visual effects and background music
- Player boundaries

Controls:
WASD to move the fighter jet
Space to shoot lasers
